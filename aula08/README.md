# Como o Prompt Ajudou?

Este projeto foi desenvolvido utilizando uma abordagem de Co-Creation, onde os prompts foram utilizados para estruturar a lógica de organização de sistemas de arquivos e o consumo de APIs externas. A IA auxiliou na implementação de tratamento de erros robustos (como o try-except no gerenciamento de arquivos) e na sanitização de dados de entrada para a consulta de CEPs, garantindo que o código fosse resiliente a falhas de rede e entradas inválidas do usuário.

# Análise Crítica

A colaboração humano-IA neste projeto demonstrou ser altamente eficaz na automação de tarefas repetitivas, como a categorização de extensões e a estruturação de protocolos de segurança, como o uso de diretórios temporários para testes. Enquanto a IA acelerou significativamente a prototipagem e sugeriu tratamentos de erro robustos para o consumo da API ViaCEP, a supervisão humana foi essencial para validar regras de negócio específicas e prever limitações de ambiente que a ferramenta não alcança sozinha. Essa sinergia equilibra a produtividade tecnológica com o olhar crítico necessário para garantir que o código não seja apenas funcional, mas também seguro e otimizado para cenários reais.
