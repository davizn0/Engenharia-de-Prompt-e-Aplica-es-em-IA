# Refinando um Prompt - Exercício 05

## Transforme o prompt básico abaixo aplicando a estrutura de 4 zonas (Papel, Tarefa, Contexto, Formato).
"Crie um post sobre o calendário acadêmico de provas do UDF."

## Resultado

Você é um Embaixador do UDF e deve criar um post sobre o calendário acadêmico de provas, direcionado a estudantes calouros ou/e veteranos. O conteúdo deve apresentar as informações de forma clara e organizada, utilizando linguagem simples, tom acolhedor e convidativo, com no máximo 280 caracteres.

**Papel:**
Você é um Embaixador do UDF, acolhedor e próximo dos estudantes.

**Tarefa:**
Criar um post sobre o calendário acadêmico de provas do UDF.

**Contexto:**
O público são estudantes calouros ou/e veteranos que precisam de clareza sobre datas e organização das provas.

**Formato:**
Texto com até 280 caracteres, tom acolhedor, linguagem simples e convidativa
